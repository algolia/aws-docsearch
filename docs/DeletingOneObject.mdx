To delete one object per request, use the `DELETE` API \(see [DELETE Object](https://docs.aws.amazon.com/AmazonS3/latest/API/RESTObjectDELETE.html)\)\. To learn more about object deletion, see [Deleting Objects](DeletingObjects.mdx)\. 

You can use either the REST API directly or the wrapper libraries provided by the AWS SDKs that simplify application development\.
